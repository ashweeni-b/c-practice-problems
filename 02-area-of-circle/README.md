## Area of Circle

### Problem Statement
Write a program to calculate the area of a circle whose radius is given.

---

### Input
A single line containing a floating value N.

### Output
A single line containing area of circle N.

---

### Test Case 1
Input: 5.0 <br>
Output: 78.5398 <br>

### Test Case 2
Input: 3.0 <br>
Output: 28.2743 <br>

---

### Algorithm
1. Input `r`.
2. Calculate `area = 3.1415 * r * r`.
3. Output `area`.
