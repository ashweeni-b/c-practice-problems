## Cube of a Number

### Problem Statement
Take a number (n) from user and output its cube (n * n * n).

---

### Input
A single integer N representing the number whose cube needs to be calculated.

### Output
Print a single integer - the cube of N.

---

### Test Case 1
Input: 2 <br>
Output: 8 <br>

### Test Case 2
Input: 5 <br>
Output: 125 <br>

---

### Algorithm
1. Input `N`.
2. Calculate `cube = N * N * N`.
3. Output `cube`.
