## Multiplication Table

### Problem Statement
Take a number (num) from user and output the multiplication table of the number.

---

### Input
A single integer num representing the number whose multiplication table is to be printed

### Output 
Print the multiplication table of the num in the following format,
`4 x 6 = 24`

---

### Test Case 1
Input: <br>
Enter the value of num: 4 <br>

Output: <br>
Table of 4: <br>
4 x 0 = 0 <br>
4 x 1 = 4 <br>
4 x 2 = 8 <br>
4 x 3 = 12 <br>
4 x 4 = 16 <br>
4 x 5 = 20 <br>
4 x 6 = 24 <br>
4 x 7 = 28 <br>
4 x 8 = 32 <br>
4 x 9 = 36 <br>
4 x 10 = 40 <br>

### Test Case 2

Output: <br>
Table of 2: <br>
2 x 0 = 0 <br>
2 x 1 = 2 <br>
2 x 2 = 4 <br>
2 x 3 = 6 <br>
2 x 4 = 8 <br>
2 x 5 = 10 <br>
2 x 6 = 12 <br>
2 x 7 = 14 <br>
2 x 8 = 16 <br>
2 x 9 = 18 <br>
2 x 10 = 20 <br>

---

### Algorithm 
1. Input number `num`.
2. Initialize `i` to 0.
3. Until `i <= 10`, print the multiplication table in the format as `4 x 6 = 24`.
4. Increment `i` by 1.
