## Area of Square

### Problem Statement
Write a program to calculate the area of a square whose side is given.

---

### Input
A single integer N representing the side of square.

### Output
Print a single integer - the area of square with side N.

---

### Test Case 1
Input: 5 <br>
Output: 25 <br>

### Test Case 2
Input: 9 <br>
Output: 81 <br>

---

### Algorithm
1. Input `N`.
2. Calculate `area = N * N`.
3. Output `area`.
